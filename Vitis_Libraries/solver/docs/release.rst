Release Note
============

.. toctree::
   :hidden:
   :maxdepth: 1

2022.1
------

In this relese, we migrate legacy API from Vivado_HLS to solver library. They're all hls::stream based API and support std::complex type.

* Cholesky
* Cholesky Inverse
* QR Inverse
* QRF (QR decomposition)
* SVD
